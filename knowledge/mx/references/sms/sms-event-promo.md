# SMS: Event Promo

Event Promo is the SMS that ties a promotion to a real moment: a holiday, a cultural occasion, an awareness day, a brand anniversary. Use it when the event genuinely fits the brand and the offer has an obvious connection to the moment. Use it when the moment alone gives the send a reason to exist. Do not use it for forced tie-ins where the brand has nothing to say about the event, and do not pile on every holiday in the calendar; an off-brand event SMS dilutes the channel. Playbook rule: do not force the event if it is not on-brand. Event plus offer fit naturally together. Tie-in quick.

## Structural rules

- Length: under 160 characters total, including link.
- Opening: one short beat connecting the event to the audience or product. Not a history lesson about the event.
- Body: state the offer mechanic and the code if any. Codes uppercase.
- CTA: one. Shopping verb plus a shortened link.
- Sign-off: none.
- Compliance: links shortened; STOP handled by the platform.

## When the brief should call it

```
**Campaign Direction**: <one sentence naming the event and the offer that goes with it>

**Offer**: <exact mechanic, code, expiration tied to the event window>

**Links**: <single shortened destination matching the offer>

**Message Type**: Event Promo
```

The brief carries the event tie-in language and the offer terms verbatim. The copywriter does not invent the connection between the brand and the event.

## Example structural fragment

```
<ONE-LINE EVENT TIE-IN>. Extra <EXACT OFFER> <EVENT NAME> - use code <EXACT CODE> at checkout. Shop now: [SHORT_LINK]
```

If the message spends most of its characters explaining the event, the type has drifted. Trim the tie-in to one beat and let the offer carry the rest.
