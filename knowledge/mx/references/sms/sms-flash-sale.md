# SMS: Flash Sale

Flash Sale is the most common SMS type. Use it when the send's whole job is landing a time-boxed offer: today-only deals, first-X-buyer gifts, end-of-day extensions, and short-fuse drops. Use it only with real deadlines or real scarcity. Do not invent urgency; readers notice and future urgency loses power. Playbook rule: lead with time pressure or scarcity, not setup. Show the code when there is one. Keep it punchy.

## Structural rules

- Length: under 160 characters total, including link. SMS hard rule across every type.
- Opening: time pressure or scarcity in the first beat ("Today only", "Final hours", "First 100 only"). No greeting, no brand-as-narrator preamble.
- Offer statement: explicit. The exact discount mechanic and the code if there is one. Codes uppercase, no decoration.
- CTA: one. Verb plus a shortened link. Never two links, never a link plus a phone number, never a CTA branching to "or reply STOP".
- Sign-off: none. SMS does not get a sign-off.
- Compliance: STOP keyword is handled by the sending platform; do not bake "Reply STOP" into the message body unless the brand mandates it. Links must be shortened.

## When the brief should call it

```
**Campaign Direction**: <one sentence naming the deadline or scarcity hook as the campaign job>

**Offer**: <exact mechanic, code, expiration window>

**Links**: <single shortened destination matching the offer>

**Message Type**: Flash Sale
```

The brief carries deadline and code verbatim. The copywriter does not invent percentage, window, or code. Missing terms become `missing info: <what is missing>` in the brief.

## Example structural fragment

```
<TIME-PRESSURE OPENER>: <EXACT OFFER FROM BRIEF>. Use code <EXACT CODE>. Shop now: [SHORT_LINK]
```

If the message opens with brand backstory or "we wanted to let you know," it has drifted. Cut back to deadline and code.
