# SMS: Win-Back

Win-Back is the SMS for reopening the relationship with a lapsed subscriber or buyer. Use it inside an SMS winback sequence triggered by a winback-eligible segment, or as a one-off reactivation push when the brand has a concrete reason to invite the reader back. Use it when the offer is the lever and the message can stay short and inviting. Do not use "we miss you" framing; in MX winback copy across channels it is forbidden, and sentiment alone has been shown to underperform for a list that has already ignored months of campaigns. Do not use this type for general acquisition. Playbook rule: gentle reactivation over guilt. Return reason concrete. Short and inviting.

## Structural rules

- Length: under 160 characters total, including link.
- Opening: invite, do not accuse. "It's time", "Come back to", a forward-looking beat. Never "we miss you", never "where have you been", never any guilt framing.
- Body: state the concrete return reason. Offer mechanic plus code if there is one. Codes uppercase.
- CTA: one. Shopping verb plus a shortened link. "Shop now", "Use my code".
- Sign-off: none.
- Compliance: links shortened; STOP handled by the platform.

## When the brief should call it

```
**Campaign Direction**: <one sentence naming the concrete reactivation lever>

**Offer**: <exact mechanic, code, expiration>

**Links**: <single shortened destination matching the offer>

**Message Type**: Win-Back
```

The brief carries the offer terms and the code verbatim. The copywriter does not invent a percentage, a code, or a "we miss you" angle.

## Example structural fragment

```
<FORWARD-LOOKING OPENER> with <EXACT OFFER>. Use code: <EXACT CODE>. Shop now: [SHORT_LINK]
```

If the message starts apologizing for sending or begging the reader back, the type has drifted. Cut the sentiment and let the offer carry the invite.
