# Image Block Section

Image Block is a visual-led support block. The image does the persuasion work; copy stays minimal and serves the visual. Use it when the campaign has a real image-led move to make: a before/after, a split frame, a configurator screenshot, a product-in-context shot that explains itself faster than words could. Use it as a SHORT / scan-first section, often between two heavy blocks to give the reader breathing room. Do not use it as a placeholder for not-yet-decided creative; if the image's role is not clear, the section is not earning its place.

## Structural rules

- Required: one image with a clearly named role. Not a generic product photo, not a stock image. A specific visual doing a specific job.
- Required: a short label or heading describing the visual's role (e.g. "CHAOS VS. CONTROL", "BEFORE / AFTER", "THE FIT").
- Optional: a one-line caption framing the image. Maximum one sentence.
- Optional: a CTA below the image if it leads somewhere specific.
- Copy total: 1 to 3 short lines, including label and caption. Anything longer turns the section into something else.
- The image briefing token sits in the section so the copywriter can hand off a clear image direction to the designer. Use `[image: <plain description of what the visual must show>]` as the canonical placeholder shape.

## Placeholder shape in the brief

```
**Email Sections**:
1. Hero Section
2. <heavy section>
3. Image Block
4. <next canonical section label>

**Design Notes**: <factual description of the image's required content, only if the brief carries it>
```

Image Block sits inside a 4-section DESIGNED email when the other three include a heavy block; it adds visual rhythm without adding word count. The brief carries factual design notes; it does not pre-write the caption.

## Example structural fragment

```
## <SHORT HEADING OR LABEL: 2-5 words naming the visual's role>

[image: <PLAIN DESCRIPTION of what the visual must show, including any split-frame or before/after framing the brief specified>]

<OPTIONAL CAPTION: one sentence framing the image, only if the visual needs a verbal anchor>

[<OPTIONAL CTA VERB PHRASE>](<verified link from brief>)
```

A split-frame image gets a comparison label ("CHAOS VS. CONTROL"). A before/after gets a transformation label ("BEFORE / AFTER", "DAY 1 vs DAY 30"). A product-in-context shot gets a scenario label ("THE FIT", "IN THE WILD"). Pick the label that matches the visual's actual job.
